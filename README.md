# Trip Planner (מתכנן הטיולים האישי)
אפליקציית Web לארגון ותכנון מסלולי טיול עם תמיכה במפות, תחזית מזג-אוויר, ושמירת היסטוריה.

## הרצה מקומית
1. יצירת `.env` (ראה שלב 2)
2. הרצת השרת: `cd server && npm install && npm run dev`
3. הרצת הלקוח: `cd client && npm install && npm start`

## ארכיטקטורה
- Backend: Node.js + Express + MongoDB + JWT  
- Frontend: React + Leaflet + Axios  
- סקריפטים: Dockerfile, .env-example, ו־GitHub Actions (בעתיד)
